## System Overview

The AI Interview Assistant is a **full-stack web application** that helps users practice job interviews with real-time AI-powered feedback.

### Tech Stack

| Layer | Technology | Purpose |
|-------|------------|---------|
| **Frontend** | React 18 + TypeScript | UI & user interactions |
| **Styling** | Tailwind CSS | Responsive design |
| **State** | React Context | Auth & global state |
| **Backend** | FastAPI (Python) | REST API server |
| **AI/ML** | Sentence-BERT | Semantic similarity scoring |
| **LLM** | Google Gemini | AI coaching & feedback |
| **Speech** | Faster-Whisper | Audio transcription |
| **Database** | Supabase (PostgreSQL) | Data persistence |
| **Auth** | Supabase Auth | User authentication |

## Key Design Decisions

| Decision | Rationale |
|----------|-----------|
| **Local Whisper** | No API costs, privacy, works offline |
| **Gemini 2.0 Flash** | Best free tier (1500/day), fast, accurate |
| **6-Score System** | Comprehensive yet understandable feedback |
| **Supabase** | Free tier sufficient, great DX, built-in auth |
| **No audio storage** | Privacy first, reduces storage costs |
| **Multi-key rotation** | Handle API rate limits gracefully |
| **Session-based** | Track progress, enable resume/retry |
