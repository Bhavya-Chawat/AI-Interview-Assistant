## The Interview Process

### Step 1: Select Your Domain

Choose from 5 interview categories:

| Domain | Best For |
|--------|----------|
| üíº **Management** | Leadership roles, team leads, managers |
| üíª **Software Engineering** | Developers, engineers, tech roles |
| üí∞ **Finance** | Analysts, accountants, banking |
| üìö **Teaching** | Educators, trainers, professors |
| üõí **Sales** | Sales reps, account managers, BD |

### Step 2: Enter Job Description

Paste the job description you're preparing for. This helps:
- Generate more relevant questions
- Score your answers against job requirements
- Provide tailored feedback

**Tip:** The more detailed the JD, the better the questions!

### Step 3: Upload Resume (Optional)

Upload your resume (PDF or DOCX) to:
- Get a skills gap analysis
- See matched vs missing skills
- Receive improvement suggestions

### Step 4: Answer Questions

You'll receive 10 questions:
- **5 Behavioral** ‚Äî Past experiences, scenarios
- **5 Technical** ‚Äî Domain-specific knowledge

For each question:

1. **Read the Question Carefully**
   - Take 10-15 seconds to think
   - Plan your STAR response (if behavioral)

2. **Click "Start Recording"**
   - Speak clearly into your microphone
   - Aim for 1-2 minutes per answer
   - Maintain a steady pace

3. **Click "Stop" When Done**
   - Wait 5-10 seconds for AI analysis
   - Review your feedback

4. **Navigate Questions**
   - Use sidebar to jump to any question
   - Skip and return to questions later
   - Re-attempt questions for practice

## Understanding Your Scores

### The 6-Score System

| Score | Weight | What It Measures |
|-------|--------|------------------|
| **Content** | 30% | How relevant your answer is to the question |
| **Delivery** | 15% | Speaking pace, filler words (um, uh, like) |
| **Communication** | 15% | Grammar, vocabulary, clarity |
| **Voice** | 15% | Tone consistency and energy |
| **Confidence** | 15% | Assertiveness and composure |
| **Structure** | 10% | Organization, STAR method usage |

### Score Interpretation

| Range | Rating | What It Means |
|-------|--------|---------------|
| üü¢ **90-100** | Outstanding | Interview ready, exceptional |
| üü¢ **80-89** | Excellent | Very strong, minor polish needed |
| üü° **70-79** | Good | Solid, some areas to improve |
| üü° **60-69** | Fair | Acceptable, notable weaknesses |
| üî¥ **40-59** | Needs Work | Significant improvement needed |
| üî¥ **0-39** | Poor | Major areas require attention |

### STAR Method Analysis

For behavioral questions, you'll see a STAR breakdown:

- **S**ituation ‚Äî Did you set the context?
- **T**ask ‚Äî Did you explain your responsibility?
- **A**ction ‚Äî Did you describe what you did?

## Tips for Best Results

### Recording Tips

| Do ‚úÖ | Don't ‚ùå |
|------|--------|
| Use a quiet room | Record in noisy areas |
| Speak naturally | Rush through answers |
| Use concrete examples | Give vague responses |
| Pause to think | Fill silence with "um" |
| Face the microphone | Turn away while speaking |

### Optimal Answer Length

| Question Type | Ideal Duration |
|---------------|----------------|
| Behavioral | 90-120 seconds |
| Technical | 60-90 seconds |
| Quick intro | 45-60 seconds |

### STAR Method Template

For behavioral questions:

1. **Situation** (15 sec)
   > "In my previous role at X, we faced Y challenge..."

2. **Task** (10 sec)
   > "I was responsible for Z..."

3. **Action** (45 sec)
   > "I took the following steps: First... Then... Finally..."

4. **Result** (20 sec)
   > "As a result, we achieved... I learned..."

### Improving Your Scores

| Low Score | How to Improve |
|-----------|----------------|
| **Content** | Include more relevant keywords, address the question directly |
| **Delivery** | Practice pacing, reduce filler words |
| **Communication** | Use varied vocabulary, complete sentences |
| **Voice** | Project confidence, vary your tone |
| **Confidence** | Practice more, prepare key stories |
| **Structure** | Use STAR method, organize thoughts first |

## Troubleshooting

### "Microphone not working"

1. Click the lock icon in browser address bar
2. Set microphone permission to "Allow"
3. Refresh the page
4. Try a different browser

### "AI feedback not showing"

- API quota may be exceeded
- Wait for quota reset (midnight PT)
- Or create new API key

### "Session not saving"

1. Check your internet connection
2. Look for error messages
3. Contact support if persistent
